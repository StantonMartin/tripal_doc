
Requirements
===============

- Drupal (see supported versions below)
- Drupal core modules: Search, Path, Views, and Field.
- PostgreSQL 10+
- PHP 8 (tested with 8.0 + 8.1)
- Apache 2+
- Composer 2+
- UNIX/Linux
